## 使用介紹
### 點選下方選單'使用介紹'  
pic   



1. 查詢咖啡店，傳送「!name 咖啡廳店名」，機器人將會快速答覆店家位置及資訊。

2. 輸入「不限時」，機器人將答覆您 8 家大台北地區用餐不限時間的咖啡廳。

3. 傳送 「位置資訊」，即可查詢距離您 5 公里處的 5 家咖啡廳。
  
4. 點選選單「烘焙小知識」

5. 點選「咖啡豆小知識」

6. 點選「Metro Taipei」，連結外部台北捷運站路線圖
